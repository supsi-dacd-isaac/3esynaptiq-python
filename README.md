# 3esynaptiq-python

Script to get PV datasets from 3E Synaptiq web services and store them in an InfluxDB (https://www.influxdata.com/) DB.

**Usage:**
<pre>
python3 bridge.py -c conf/example.json
</pre>

**Requirements:**
<pre>
python>=3.5.2
jsonschema>=2.6.0
influxdb>=4.0.0
pytz>=2016.10
</pre>
